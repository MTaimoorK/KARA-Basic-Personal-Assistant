# Alexa Voice Assistant

This is a Python script that creates a voice assistant named Alexa. The assistant can perform various tasks such as:

- Greet the user based on the time of day
- Introduce itself and provide information about its creator
- Search Wikipedia for information on a given topic
- Open Google in the default web browser
- Search Google for a specific query and provide a summary
- Open any website by specifying the URL
- Search YouTube for a specific query
- Close the Google Chrome browser

## Requirements

To run this script, you need to have the following Python libraries installed:

- pyttsx3
- wikipedia
- webbrowser
- pywhatkit
- speech_recognition
- datetime
- os

You can install these libraries using pip:


pip install pyttsx3 wikipedia webbrowser pywhatkit SpeechRecognition


## Usage

1. Clone or download the repository.
2. Navigate to the project directory.
3. Run the `kara.py` script:


python kara.py


4. The assistant will greet you and prompt you to give a voice command.
5. Speak your command clearly, and the assistant will respond accordingly.
